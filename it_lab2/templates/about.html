О нашем проекте

Этот проект демонстрирует основы работы с HTTP в Python 3 с использованием стандартной библиотеки.

Технологии

  • Python 3
  • HTTP сервер из стандартной библиотеки
  • HTML5 и CSS3
  • Обработка форм

Цели проекта

Изучение фундаментальных концепций HTTP, создание веб-сервера и понимание принципов клиент-серверного взаимодействия.
